# CanvasToy.js

lightweight webgl rendering tool

###Current features:

-   Basic openrations of 3d object: translate, scale and rotate
-   Perspective and orthogonal projection types of Camera
-   Texture-mapping
-   Lambert material
-   Custom shader material

### Build && Test

```bash
npm install
./building.sh && ./testing.sh
```
